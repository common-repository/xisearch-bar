XiSearch content tools: increase the quantity and improve the quality of your site content thanks to the advanced search options on every page of your site

== Description ==

XiSearch bar is designed to increase useful content on your site by adding your own search engine, including onsite search as well as Internet search, search of videos, images, Wikipedia articles and adding text translator option.

XiSearch provides the variety of color themes, changeable location on a page and selection of different bar languages.

Quick search option gives opportunity to get any useful information without unnecessary clicks.

XiSearch Content hint is a part of XiSearch tools goes together with the bar but it can be also installed and used separately. 

Please, refer to the details of how to choose and get XiSearch bar on http://www.xisearch.com/


== Installation ==

1. Upload plugin zip file to the `/wp-content/plugins/` directory.
2. Activate the plugin. The search bar will appear on your website immediately with default parameters: search in English, black/blue theme, position - in the bottom of the page. You have to go the next step to change the default settings.
3. Express setting: Go to `Appearance`->`XiSearch Configuration`, select `Express settings` and change XiSearch bar view. Settings are limited to 4 themes (black&blue, light gray, light green, light red), 2 positions (top and bottom), 4 languages (English, Spanish, German and Russian)
4. Full set of settings: Register to xisearch.com, create the bar for your site and get a script for CMS. During creation you can choose between different languages, 42 themes and select flexible position on the page. After that go to `Appearance`->`XiSearch Configuration` and paste the script in the field `XiSearch script for Wordpress`.
